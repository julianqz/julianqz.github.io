Throughout my PhD, I was fortunate to be part of a team in my lab at Yale that developed and maintains Immcantation, an open-source software suite for conducting start-to-finish analysis of antibody sequences.

The Immcantation suite consists of several packages/libraries, some in Python and some in R.

I mainly contributed to two R packages -- SHazaM and Alakazam.

Here are some snapshots of me in action.

  • Updating a core function in Alakazam
  • A minor bug fix in Alakazam
  • Rewriting a core function and adding tests correspondingly in Alakazam
  • Tracking a user-reported bug in Issues and fixing the bug in SHazaM
  • Adding a vignette to SHazaM. The vignette can be viewed in the SHazaM doc.
  • Resolved an issue request for new feature in SHazaM
  • Another feature request in SHazaM
  • Resolved an issue request for adding tests in SHazaM
  • More testing for SHazaM
